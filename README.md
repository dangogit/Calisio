# Calisio

Calisthenics workout tracker with exercise planning, timers, and progress history.

## Tech Stack

- React Native, Expo SDK 52
- Expo Router (file-based navigation)
- TypeScript
- Zustand (state management)
- NativeWind (Tailwind for React Native)
- React Native Reanimated

## Features

- Create and manage custom workout plans
- Track exercises with sets, reps, and superset support
- Circular countdown timer for rest periods
- Log past workouts with full history
- Edit and update existing workouts
- Upload workout data via document picker
- Themed UI with design tokens

## Getting Started

```bash
git clone https://github.com/dangogit/Calisio.git
cd Calisio
npm install
npx expo start
```
